Этот проект представляет собой Telegram-бота для приёма документов и автоматического преобразования их содержимого в обычный текст.  
Бот поддерживает форматы **TXT, MD, DOCX, PDF** и предназначен для дальнейшей обработки текста — суммаризации, извлечения ключевых слов и построения базы знаний.

## Основные возможности

- Приём документов через Telegram
- Конвертация файлов в текстовый формат (`.txt`)
- Сохранение файлов в отдельную папку
- Поддержка PDF и документов Microsoft Word
- Выжимка из нескольких предложений 3-5 создание summary
- Выписывает основные слова key_words
- Записывает всю информацию в Google Sheets

- Возможность масштабирования под серверные решения
